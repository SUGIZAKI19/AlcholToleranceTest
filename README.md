たったの13項目の質問に答えるだけで，お酒の強さが数値化されるスカウター的なもの。得点が０より大きければ酒豪で，マイナスなら下戸と判定されます。

# 🍻 Project setup
## プロジェクトへの参入

```
git clone https://github.com/SUGIZAKI19/AlcoholToleranceTest
```

## npm 持ってない人向け
1.  npmのインストール
    ```
    npm install
    ```

2.  ローカルサーバーの立ち上げ
    ```
    npm run serve
    ```

3.  Compiles and minifies for production
    ```
    npm run build
    ```
