# 🍻 Project setup
## プロジェクトへの参入

```
git clone https://github.com/SUGIZAKI19/AlcoholToleranceTest
```

## npm 持ってない人向け
1.  npmのインストール
    ```
    npm install
    ```

2.  ローカルサーバーの立ち上げ
    ```
    npm run serve
    ```

3.  Compiles and minifies for production
    ```
    npm run build
    ```
